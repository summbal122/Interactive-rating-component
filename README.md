# Rating Feedback App
This is a simple web application that allows users to provide feedback through a rating system. Users can select a rating from 1 to 5 and see their selection on a thank you page.

## Features
- Responsive design with a clean and modern layout.
- Allows users to select a rating and store it using localStorage.
- Displays a thank you message with the selected rating on the confirmation page.

## Technologies Used
- HTML
- CSS
- JavaScript.
